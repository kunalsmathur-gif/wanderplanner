# WanderPlan Test Recordings

Store your screen recordings here.

## Naming Convention:
- `YYYY-MM-DD_test-scenario-name.mp4`
- Example: `2026-06-17_full-user-journey.mp4`

## Recommended Settings:
- Format: MP4 (H.264)
- Resolution: 1920x1080 or native screen
- Frame rate: 30 fps
- Audio: Include microphone for narration

## After Recording:
1. Review the video
2. Note any bugs with timestamps
3. Update E2E_TESTING_GUIDE.md with results
